## 🎯 **DEPLOYMENT COMMANDS**

### **Quick Deploy**
```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy to production
vercel --prod
```

### **Using Deploy Script**
```bash
# Make script executable
chmod +x deploy.sh

# Run deploy script
./deploy.sh
```

### **Test Locally First**
```bash
# Start local server
./start-local.sh

# Or manually
python3 -m http.server 8000
```
